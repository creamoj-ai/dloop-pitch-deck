## 🎯 Generare il PDF: Metodo Rapido (1 minuto)

### **Metodo 1: Tramite Pulsante nel File**

1. **Apri il file HTML nel browser:**
   ```
   Clicca due volte su: COSTI_CLIENTE_FINALE_SEMPLICE.html
   ```

2. **Cerca il pulsante in alto:**
   - Vedrai un pulsante blu: **"🖨️ Stampa / Salva come PDF"**

3. **Clicca il pulsante**
   - Si aprirà automaticamente la finestra di stampa

4. **Scegli "Salva come PDF"**
   - Destinazione: **"Salva come PDF"**
   - Clicca **"Salva"**

5. **Nomina il file:**
   - Nome consigliato: `DLOOP_Costi_Cliente_Finale.pdf`
   - Salva in questa cartella: `dloop-pitch-deck/pdf/`

---

### **Metodo 2: Keyboard Shortcut (Ancora più veloce)**

1. **Apri il file nel browser**
2. **Premi:** `Ctrl + P` (Windows) o `Cmd + P` (Mac)
3. **Scegli "Salva come PDF"**
4. **Clicca "Salva"**

---

### **Metodo 3: Chrome DevTools (Pro)**

1. **Apri il file in Google Chrome**
2. **Premi:** `F12` (apre Developer Tools)
3. **Premi:** `Ctrl + Shift + P` (apre Command Palette)
4. **Digita:** `Print to PDF`
5. **Premi Enter**
6. **Configurazioni ottimali:**
   - Background graphics: **ON** (per colori)
   - Margins: **Default**
   - Paper size: **A4**

## 🔧 Troubleshooting

### **Problema: I colori non si vedono nel PDF**
**Soluzione:** Assicurati di aver attivato "Grafica di sfondo" nelle impostazioni di stampa

### **Problema: Le tabelle sono tagliate**
**Soluzione:** Usa margini "Normale" (non "Ridotto") e orientamento "Verticale"

### **Problema: Il file è troppo pesante**
**Soluzione:** I PDF DLOOP sono <5MB, dovrebbe andare bene su WhatsApp

### **Problema: Non riesco a trovare il pulsante di stampa**
**Soluzione:** Premi semplicemente `Ctrl+P` per aprire la finestra di stampa

## 💡 Pro Tips

### **Tip 1: Zoom prima di stampare**
Se il testo è troppo piccolo, premi `Ctrl ++` per zoommare prima di stampare

### **Tip 2: Preview prima di salvare**
Sempre clicca "Anteprima" nella finestra di stampa per vedere come appare

### **Tip 3: Condivisione multipla**
Se vuoi mandare a più persone:
1. Genera il PDF una sola volta
2. Condividilo tramite WhatsApp, Email, Drive, ecc.

### **Tip 4: Watermark personalizzato**
Se vuoi aggiungere "CONFIDENZIALE" o una data:
1. Genera il PDF normalmente
2. Usa un editor PDF gratuito (es: ilovepdf.com) per aggiungere filigrana
