📖 Come Generare i PDF

✅ Metodo Rapido (1 minuto)

  1. Clicca uno dei bottoni "🖨️ Genera PDF" sopra
  2. Si aprirà il file HTML nel browser
  3. Premi Ctrl+P (Windows) o Cmd+P (Mac)
  4. Scegli "Salva come PDF"
  5. Attiva "Grafica di sfondo" per i colori
  6. Clicca "Salva" in questa cartella

⚠️ Impostazioni Importanti

  • ✅ Grafica di sfondo: ATTIVATA (per colori vividi)
  • ✅ Formato carta: A4
  • ✅ Orientamento: Verticale
  • ✅ Margini: Normale
